# Blog Platform with Comments

## Overview
This is a full-stack Blog Platform developed using HTML, CSS, JavaScript, Node.js, Express.js, and MySQL.
Users can create, manage, and interact with blog posts through comments. The project demonstrates content management and user interaction features commonly found in blogging applications.

## Features
- User Registration and Login
- Authentication System
- Create Blog Posts
- Edit Blog Posts
- Delete Blog Posts
- Comment System
- RESTful APIs
- MySQL Database Integration

## Technologies Used
Frontend: HTML, CSS, JavaScript

Backend: Node.js, Express.js

Database: MySQL

## Expected Outcome
Users can publish blog posts, manage content, add comments, and interact with other users through a simple blogging platform.
